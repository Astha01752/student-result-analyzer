# Student Result Analyzer

A simple Python program that calculates the **total marks**, **percentage**, **grade**, and **class topper** from user-entered student data.  
This project is created for the Vityarthi "Build Your Own Project" evaluation under Computer Engineering (Python).

---

##  Project Overview

This project allows the user to:

- Enter subject names  
- Enter multiple students' names and marks  
- Automatically calculate:  
  - Total marks  
  - Percentage  
  - Grade  
- Identify and display the class topper  
- Show results in a clean tabular format in the console

It removes the need for manual result calculation and reduces errors.
