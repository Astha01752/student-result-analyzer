## Problem Statement
Manual result calculations for multiple students are time-consuming and error-prone. Teachers need a simple program to calculate totals, percentages, grades, and topper automatically.

## Scope of the Project

- Accept multiple students  
- Validate marks  
- Calculate totals, percentages & grades  
- Identify class topper  
- Display formatted table output  

## Target Users
- Teachers  
- Students  
- Coaching centers  
- Anyone who needs quick result analysis  

## High-Level Features
- Dynamic input  
- Auto calculations  
- Grade assignment  
- Topper detection  
- Clean console output
